#  University Dormitory & Restaurant Management System (UDRMS)

##  Description
A C++/Qt application for managing university dormitories, rooms, students, and restaurants.

##  Features
-  **Student Management** (Add/Remove/Assign students)
-  **Dormitory Management** (Add dormitories and rooms)
-  **Restaurant Management** (Set and view daily menus)
-  **Data Persistence** (Save/Load data to files)
-  **Modern GUI** with dark theme and icons

##  Technologies Used
- **C++17**
- **Qt 6** (Widgets)
- **CMake** (Build system)
- **Git/GitHub** (Version control)
